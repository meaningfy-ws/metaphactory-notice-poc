Application capabilities overview

The application is designed for procurement notices data. It's organized around the two types of data: Notice (epo:Notice) and organization (org:Organization). It allows following operations on notice and organization data:
  1. View a resource page
  2. Browse
  3. Search by facets
  4. Search by a keyword
  5. Navigate between related resource pages (for example visit a buyer page)

Moving around the application

The application adheres to the common practice of leveraging hyperlink for the purposes of navigating from one resource to another.

Throughout the application, following kinds of resources are accessible via a hyperlink:
  1. Underlying Knowledge Graph resources
    1. Internal resources having a dedicated details page (Notice or Organization instances; example)
    2. Internal resources without a dedicated page (example)
    3. Supported vocabulary terms (example)
    4. ePO ontology terms (example)
  2. Wikidata (wikidata.metaphacts.com) pages (example)
  3. External web pages (example)

Display details of a resource

Notice and Organization data are presented on dedicated resource details pages. See exemplary notice and organization pages.

Navigate between notices and organizations

Both notice and organization details pages includes proper sections holding references to a related notice or organization as hyperlinks.

Clicking on such a hyperlink opens a page of the related resource.

Preview a resource

The feature displays a preview of an RDF resource containing properties with their values. The preview is a pure view of underlying KG subgraph and lets a more experienced user to dig into the linked data. The function is available on search list page as well as resource (notice / organization) detail pages.

Term definition lookup

Selected sections of a resource details offer an additional description of a section available after hovering accompanying help icon. These descriptions are definitions taken from EPO ontology.

Events timeline

The notice details page offers a view containing dates of important events marked on a timeline.

Map

The organization details page contains a map presenting an approximate location (based on either NUTS or country code) of the organization headquarters.

There is a tooltip attached to a map marker which shows a link to the public Wikidata portal developed by Metaphacts which is useful for further exploration. The page also contains a link to a Wikidata resource page itself.
