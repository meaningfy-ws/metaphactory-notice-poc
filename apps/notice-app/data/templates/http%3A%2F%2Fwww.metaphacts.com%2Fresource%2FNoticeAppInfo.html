How to use this application

Application features

The application is designed for procurement notices data. It's organized around the two types of data: Notice (epo:Notice) and organization (org:Organization). It allows following operations on notice and organization data:
  1. View a resource page
  2. Browse
  3. Search by facets
  4. Search by a keyword
  5. Navigate between related resource pages (for example visit a buyer page)

Features

Browse notice/organization resources

Narrow down search results by using facets

Narrow down search results by searching for a keyword

Limitation: the keyword search is based on exact match of words.

Display details of a resource

Preview a resource

The feature displays a preview of an RDF resource containing properties with their values. The preview is a pure view of underlying KG subgraph and lets a more experienced user to dig into the linked data.

Search list view

Resource details view

Term definition lookup

Events timeline

The notice view page offers a view containing dates of important events marked on a timeline.
